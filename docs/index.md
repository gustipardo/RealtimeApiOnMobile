## Migration Considerations

### Critical Issues to Solve

1. **UI/Voice Race Conditions** - Implement state machine (FSM)
2. **Incomplete Feedback Loop** - Force answer reveal in code
3. **Zombie Sessions** - Explicit stream cleanup
4. **Excessive Hinting** - Gate in code, not just prompts

### Mobile Adaptations Needed

| Web | Mobile (Expo) |
